Мини описание сборки¶

Эта сборка основана на Calculate Linux, сборка наследует стандартный профиль calulcate:desktop.

В качестве DM используется LightDM, а качестве WM используется openbox.

Как и в родителе, для получения прав администратора системы, запущенной с LiveCD, используется логин root с паролем root. Доступ к рабочему столу выполняется пользователем guest с паролем guest.

Список используемых клавиатурных сочетаний:¶

Сочетания Openbox:¶

  • Ctrl+Alt+Left - Переключение на виртуальный рабочий стол влево
  • Ctrl+Alt+Right - Переключение на виртуальный рабочий стол вправо
  • Shift+Alt+Left - Переместить активное приложение на виртуальный рабочий стол слева
  • Shift+Alt+Right - Переместить активное приложение на виртуальный рабочий стол справа
  • Win+[1-4] - Переключить на виртуальный рабочий стол №[1-4]
  • Alt+Tab - Переключить на следущую задачу
  • Alt+Shift+Tab - Переключить на предыдущую задачу
  • Alt+F2 - Диалог запуска программ: gmrun
  • Win+e - Запуск обозревателя файлов: pcmanfm
  • Win+w - Запуск Интернет обозревателя: firefox
  • Win+t - Запуск консольки: urxvt
  • PrtScr - Запуск скриншотера для всего экрана
  • Alt+PrtScr - Запуск скриншотера для активного окна

Немного о Firefox:¶

В представленной сборке Firefox используется совместно с неоднозначным плагином, удобным прежде всего мне, Vimperator, позволяющим использовать vim-like возможности.

  • h, j, k, l, gg, G, 0, $, ZZ, ZQ - стандартные клавиши вимера
  • o/O - открыть в текущем окне ссылку / с редактированем текущей
  • t/T - открыть в новой вкладке ссылку / с редактированем текущей
  • p/P - открыть ссылку из буфера обмена в новой/текущей вкладке
  • Ctrl+o/Ctrl+i - Назад/Вперед в текущем окне
  • gt/Ctrl+n - переход на следущую вкладку
  • gT/Ctrl+p - переход на предыдущую вкладку
  • g^/g$ - переход на первую/последнюю вкладку
  • d - закрыть текущую вкладку
  • r/R - перегрузить текущую вкладку с использованием кеша / без использования кеша
  • g+ctrl+g - информация о текущей странице
  • y/Y - Скопировать урл текущей страницы/текущее выделение в буфер обмена
  • :help - Посмотреть помощь :)
